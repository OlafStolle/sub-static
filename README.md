
# DLKarte - Digitale Karten für Dienstleistungen

Eine Anwendung zur Visualisierung von Dienstleistungsstandorten und Subregionen auf interaktiven Karten.

## Funktionen

- Login-System mit Benutzerauthentifizierung
- Upload und Verarbeitung von DL-Datendateien (Excel)
- Visualisierung von Standorten und Subregionen auf interaktiven Karten
- Filterfunktionen nach verschiedenen Kriterien
- Farbliche Hervorhebung nach Gewerken, Status und anderen Attributen

## Technologien

- HTML5, CSS3, JavaScript
- Leaflet.js für Kartenvisualisierung
- IndexedDB für clientseitige Datenspeicherung
- Bootstrap für UI-Komponenten
- XLSX.js für die Verarbeitung von Excel-Dateien

## Installation und Start

1. Repository klonen
2. Webserver starten (z.B. mit Live Server oder ähnlichen Tools)
3. Die Anwendung über `index.html` im Browser öffnen
